# React Pomodoro Timer Project

This is a Pomodoro Timer project to practice react fundamentals

# How to Run Application

- Please type npm install then npm run dev

# Concepts and React Fundamentals

- Vite
- Typescript
- Context API
- React Router DOM
- React Hook Form
- Zod Validation
- Styled Components
- Local Storage
